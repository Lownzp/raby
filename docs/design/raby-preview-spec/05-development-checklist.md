# Development Checklist

本清单用于把预览图规格转成 Flutter 开发任务。执行顺序以「先组件系统，再页面闭环，再截图评审」为原则。

## 0. 开发前准备

- [ ] 确认最终 8 张预览图已经只归档在 Raby 项目目录内。
- [ ] 确认本目录规格文档是当前唯一 UI 还原口径。
- [ ] 确认贴纸规则：每页重点贴纸 1-3 个，碎装饰 0-2 处。
- [ ] 确认功能图标和贴纸分层，不把列表图标做成贴纸。
- [ ] 确认目标验收视口：360 x 780、390 x 844、430 x 932。

## 1. Theme 和 Design Tokens

建设内容：

- [ ] 建立 Raby 色彩 token：背景、卡片、主按钮、深棕文本、浅棕文本、状态色。
- [ ] 建立文字层级：页面标题、卡片标题、正文、辅助文字、数字大字。
- [ ] 建立圆角 token：小控件、输入框、卡片、底部 Tab。
- [ ] 建立阴影 token：卡片、贴纸、浮动图标按钮。
- [ ] 建立间距 token：页面边距、卡片内边距、组件间距。

验收：

- [ ] 同类颜色、圆角、阴影在页面间一致。
- [ ] 不出现随机色值和随机圆角。

## 2. 通用组件

建设内容：

- [ ] `RabyPageScaffold`：统一背景、安全区、滚动容器。
- [ ] `RabyTopBar`：返回、标题、右侧操作。
- [ ] `RabyBottomTabs`：首页、体重、我的。
- [ ] `RabyIconButton`：圆形图标按钮。
- [ ] `RabyPrimaryButton` 和 `RabySecondaryButton`。
- [ ] `RabyCard`：统一卡片外观。
- [ ] `RabyPetInfoCard`：头像、名字、次级信息、右侧操作。
- [ ] `RabySegmentedControl`：周期、分类、性别。
- [ ] `RabyTagChip`：状态、分类标签。
- [ ] `RabyTextField`：单行和多行输入。
- [ ] `RabyPhotoGrid`：相册和详情图片布局。
- [ ] `RabySticker`：限制位置、尺寸、装饰层级。

验收：

- [ ] 组件在 360px 宽屏不挤压、不竖排。
- [ ] 组件有 pressed、disabled、loading 或 selected 等必要状态。

## 3. 素材接入

建设内容：

- [ ] 建立素材目录：兔子照片、贴纸、空状态。
- [ ] 接入真实兔子照片：头像、主视觉、动态、相册。
- [ ] 接入首批贴纸：胡萝卜、体重秤、日记本、相机、小兔、花朵、叶子。
- [ ] 功能图标统一为线性图标。
- [ ] 图片加载失败时使用稳定占位。

验收：

- [ ] 图片裁切不遮挡兔子脸部。
- [ ] 相册照片色温和风格接近。
- [ ] 贴纸不遮挡核心文本。
