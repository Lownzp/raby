# 待替换素材清单

除首页外的预览页在首轮实现中使用固定比例的空白图片槽位。所有槽位由
`RabyImageSlot` 承载，替换图片时不得改变外层尺寸、圆角或页面布局。

| 页面 | 槽位 | 建议比例 | 状态 |
| --- | --- | --- | --- |
| 记体重 | 宠物头像 | 1:1 | 待替换 |
| 写日记 | 宠物头像、上传缩略图 | 1:1 | 待替换 |
| 日记详情 | 宠物头像、正文图片 | 1:1 / 4:3 | 待替换 |
| 体重趋势 | 宠物头像、体重秤插图 | 1:1 | 待替换 |
| 照片相册 | 宠物头像、相机插图、三列照片 | 1:1 / 4:5 | 待替换 |
| 我的 | 主人头像、宠物主图 | 1:1 / 4:3 | 待替换 |
| 首次建档 | 页面主视觉、宠物头像 | 4:3 / 1:1 | 待替换 |

用户通过 App 选择的头像和日记照片属于运行时数据，文件有效时仍然正常展示；
只有预制演示素材和缺图状态使用空白槽位。
