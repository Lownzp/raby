# 兔兔的诗

Raby 首页空状态文案候选集。

## 兔兔的声音

### 像兔兔在说话

**标题**

> 今天也和你在一起

**说明**

如果米粒会写日记，它大概会先写下这一句。

**按钮**：替它写下来

### 具体而克制

**标题**

> 别让回忆只剩下“它以前很可爱”

**说明**

记下它今天吃了多少草、做了什么傻事，又在哪里睡着。

**按钮**：把今天写具体

## 日常的形状

### 只在今天出现

**标题**

> 今天的我，只在今天出现

**说明**

耳朵的方向、趴下的位置，明天也许就不一样了。

**按钮**：记住今天的我

### 完整的一天

**标题**

> 一只兔兔的日子，看起来总是差不多

**说明**

吃饭、睡觉、发呆。可每一天，都是它完整的一天。

**按钮**：写下这一天

## 写给时间

### 给日子起名字

**标题**

> 给今天起一个名字

**说明**

也许是“米粒在窗边睡了很久的那一天”。

**按钮**：命名今天

### 寄给未来

**标题**

> 把今天借给未来看一眼

**说明**

让未来的你，知道此刻的米粒是什么样子。

**按钮**：寄出今天

**标题**

**按钮**：写下这些小事
